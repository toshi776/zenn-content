# はじめに

PythonとGitHub連携を使って、Zennへの自動投稿システムを作成しました。

## 背景

技術記事を複数のプラットフォームに投稿するのは手間がかかります。そこで、自動投稿システムを構築することにしました。

## 実装方法

GitHub連携を使うことで、記事ファイルをpushするだけで自動的にZennに公開されます。

## まとめ
